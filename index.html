ManyKey.org

Open source software for building custom keyboard interfaces

Keyboard Firmware (Arduino/ATmega32u4)

Arduino compatible code to power your keyboard project. Hook up momentary switches or buttons to your microcontroller, and program them to trigger whatever keyboard shortcuts you like.

Features

  • Supports boards that use an ATmega32u4, or otherwise support USB HIB with the Keyboard library
  • Supports 10+ characters pressed simultaneously per switch
  • Key configuration is updatable over USB serial (and our desktop software, coming soon)
  • Configuration stored in non-volatile memory - no battery required
