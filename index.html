ManyKey.org

Open source software for building custom keyboard interfaces

Keyboard Firmware (Arduino/Atmega)

Arduino compatible code to power your keyboard project. Hook up momentary or latching switches to your microcontroller, and program them to trigger whatever keyboard shortcuts you like.

Features

  • Supports as many switches/buttons as your microcontroller has pins
  • Supports 10+ characters pressed simultaneously per switch
  • Key configuration is updatable over USB serial (and our desktop software, coming soon)
  • Configuration stored in non-volatile memory - no battery required
